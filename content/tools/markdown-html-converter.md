# Markdown ↔ HTML Converter

Convert Markdown to HTML and HTML to Markdown instantly, right in your browser. No signup, no upload, no tracking. Built for developers who need fast, reliable conversion for READMEs, docs, blog drafts, and CMS workflows.

## What this tool does

- **Markdown → HTML**: Paste Markdown, get clean HTML with proper heading hierarchy, lists, links, and code blocks.
- **HTML → Markdown**: Paste HTML, get readable Markdown suitable for static site generators, GitHub, or Notion.
- **GitHub Flavored Markdown (GFM)**: Tables, task lists, strikethrough, autolinks, and fenced code blocks with language hints.
- **Syntax highlighting**: Code blocks render with language-aware highlighting in the preview.
- **Live preview**: See the converted output update as you type or paste.
- **One-click copy**: Copy HTML or Markdown output to your clipboard with a single button.
- **Download as file**: Save the output as a `.html` or `.md` file when you need a real artifact.
- **Keyboard shortcuts**: `Ctrl+Enter` (Windows/Linux) or `Cmd+Enter` (macOS) to run the conversion without clicking the button.
- **Mobile-friendly**: Works on phones and tablets — useful for quick edits on the go.
- **100% client-side**: Your content never leaves the browser. No server uploads, no analytics on input.

## Frequently asked questions

### Is this tool free? Do I need to sign up?

Yes, it's completely free. There's no account, no signup, and no usage limit. Open the page and start converting.

### Is my data sent to your servers?

No. The converter runs entirely in your browser using JavaScript. Your Markdown and HTML never leave your device. You can verify this by checking the network tab — there are no upload requests when you convert.

### What Markdown syntax is supported?

We support CommonMark plus GitHub Flavored Markdown (GFM) extensions: tables, task lists, strikethrough, autolinks, and fenced code blocks with language hints. Standard inline formatting (bold, italic, links, images) works as expected.

### Can I convert HTML back to Markdown?

Yes. The converter is bidirectional — paste HTML into the input panel and you'll get clean Markdown output. This is useful for migrating content from CMS platforms back into static site generators or version-controlled Markdown files.

### Does it work on mobile?

Yes. The interface is responsive and adapts to phone and tablet screens. You can paste, convert, and copy on any modern mobile browser.

### Can I download the output?

Yes. After conversion, use the download button to save the output as a `.html` or `.md` file. The file extension matches the output format.

### Is there a keyboard shortcut?

Yes. Press `Ctrl+Enter` (Windows/Linux) or `Cmd+Enter` (macOS) to run the conversion without clicking the button.

### What is GFM?

GFM stands for GitHub Flavored Markdown. It's a popular superset of CommonMark that adds features GitHub uses in READMEs, issues, and comments — most notably tables, task lists, and strikethrough. This tool supports GFM because most developers write Markdown for GitHub, docs sites, or static blogs that follow the same conventions.
